# Anastasiia Chaplyhina - Executive Assistant Portfolio

## 🎯 Project Overview
A premium, mobile-first personal portfolio/CV website designed for corporate recruiters and HR managers at Danish career fairs. The site features bilingual functionality (Danish/English) with Scandinavian minimalism design principles.

## 🌟 Key Features
- **Bilingual Toggle**: Seamless switching between Danish (DK) and English (EN) with persistent language preference
- **Mobile-First Design**: Optimized for mobile devices with responsive breakpoints for tablets and desktops
- **Scandinavian Minimalism**: Clean, premium corporate aesthetic with navy blue, dark grey, soft beige, and white color palette
- **Professional Headshot**: Corporate-style profile image with hover effects
- **Sticky Navigation**: Fixed header with language switcher and smooth scrolling navigation
- **Floating Contact Buttons**: Quick access to phone, email, and LinkedIn contact methods
- **Accessibility**: WCAG compliant with keyboard navigation and screen reader support
- **Performance**: Optimized loading with CDN resources and efficient JavaScript

## 🎨 Design Specifications
- **Color Palette**:
  - Primary Navy: `#1a2332`
  - Secondary Navy: `#2c3e50`
  - Dark Grey: `#34495e`
  - Soft Beige: `#f8f6f0`
  - Accent Blue: `#3498db`
- **Typography**: Inter font family for modern readability
- **Layout**: Mobile-first responsive design with breakpoints at 768px and 1024px
- **Icons**: Font Awesome for consistent iconography

## 📱 Technical Implementation
- **HTML5**: Semantic markup with proper ARIA labels and accessibility attributes
- **CSS3**: Modern CSS with custom properties, flexbox, grid, and animations
- **JavaScript ES6+**: Modular code with classes for language management and interactions
- **Responsive**: Mobile-first approach with media queries for tablet and desktop views

## 🎨 Customization Points
- **Colors**: Update CSS variables in `:root` selector
- **Fonts**: Change font family in CSS or HTML head
- **Content**: Modify `data-da` and `data-en` attributes in HTML
- **Images**: Replace profile image URL in hero section
- **Contact Info**: Update phone, email, and LinkedIn URLs
